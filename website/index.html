Mission

Implement the most popular machine-learning tools and offer access to their predictive capabilities as a service through a dedicated API. This abstracts the issue of designing and maintaining a custom machine-learning framework away from the customer and makes the predictive power of machine-learning more accessible, especially for small businesses and software developers.

Features

  • OCR via API
  • Spam detection via API
  • Web Interface
  • High Level language bindings (C++, Python, Javascript)
  • Create Custom Neural Networks
  • Pay for faster results
  • Push Notifications
  • Online Documentation
